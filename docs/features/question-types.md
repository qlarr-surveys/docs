# Question Types

The survey editor provides a library of question types organized into categories.

## Text-Based Questions
- **Short Text**: Single-line text input
- **Long Text**: Multi-line text area
- **Number**: Numeric input with validation
- **Email**: Email address with built-in validation
- **Multiple Short Text**: Multiple text fields in one question

## Choice-Based Questions
- **Single Choice (Radio)**: Select one option from a list
- **Multiple Choice (Checkbox)**: Select multiple options
- **Autocomplete**: Searchable dropdown
- **Single Choice Array**: Matrix with radio buttons
- **Multiple Choice Array**: Matrix with checkboxes
- **NPS**: Net Promoter Score widget

## Image/Icon Choice Questions
- **Single Icon Choice**: Choose from icon options
- **Single Image Choice**: Choose from image options
- **Single Choice Icon Array**: Icon-based matrix
- **Multiple Icon Choice**: Select multiple icons
- **Multiple Image Choice**: Select multiple images

## Date/Time Questions
- **Date**: Date picker
- **Time**: Time picker
- **Date and Time**: Combined date and time picker

## Info Elements
- **Text Display**: Display informational text
- **Image Display**: Show images
- **Video Display**: Embed videos

## Offline Capabilities
- **Barcode Scanner**: Scan barcodes using device camera
- **Capture Photo**: Take photos directly in the survey
- **Capture Video**: Record videos during survey

## Other Question Types
- **Ranking**: Rank items by preference
- **Image Ranking**: Rank images
- **File Upload**: Allow file uploads
- **Signature**: Capture digital signatures
