# Deployment

Qlarr can be deployed locally for development and testing, or on a cloud server for production use. Both options use Docker Compose to run the full stack (frontend, backend, and PostgreSQL).

- **[Local Deployment](./local)** -- Run Qlarr on your machine with a single command. Includes Mailhog for email testing and pre-configured default credentials.

- **[Cloud Deployment](./cloud)** -- Deploy to a cloud server (AWS, DigitalOcean, etc.) with a custom domain and automatic TLS via Caddy.
